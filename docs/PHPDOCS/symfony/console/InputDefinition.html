Class InputDefinition

Description
Description | Vars (details) | Methods (details)

A InputDefinition represents a set of valid command line arguments and options.

A InputDefinition represents a set of valid command line arguments and options. Usage: $definition = new InputDefinition(array( new InputArgument('name', InputArgument::REQUIRED), new InputOption('foo', 'f', InputOption::PARAMETER_REQUIRED), ));
