Class Input

Description
Description | Descendents | Vars (details) | Methods (details)

Implements interfaces:

  • InputInterface

Input is the base class for all concrete Input classes.

Input is the base class for all concrete Input classes. Three concrete classes are provided by default: * `ArgvInput`: The input comes from the CLI arguments (argv) * `StringInput`: The input is provided as a string * `ArrayInput`: The input is provided as an array
