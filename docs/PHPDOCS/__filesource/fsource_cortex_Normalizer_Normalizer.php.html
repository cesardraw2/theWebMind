  283.         /**

  294.          *               relation. It will decide it using the following
  295.          *               parameters of decision:
  296.          *               - number of properties: as many, less mergeable
  297.          *               - number of relations:  as less reffered,
  298.          *                                       more mergeable
  299.          *               - big properties:       as many big properties,

  301.          *    n:n        relations will generate an extra entity, altough,
  302.          *               if the entity with that name already exists, it
  303.          *               takes the existing one.
  304.          *               If the antity ONLY has keys, no pk will be added.
  305.          * </pre>
